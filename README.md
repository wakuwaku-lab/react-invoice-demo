# React Invoice demo (請求書)

## Features

- Editable invoice form with Japanese business format
- Automatic calculation of subtotal, tax, and total
- Professional Japanese aesthetic with Noto Serif JP typography
- Print-friendly design

## Getting Started

```bash
npm install
npm run dev
```

Open http://localhost:5173 in your browser.

## Usage

1. Edit company information in the header section
2. Enter customer details
3. Add line items with quantity and unit price
4. The system automatically calculates totals
5. Print or save as PDF

## Tech Stack

- React 18
- Vite
- Noto Serif JP / Noto Sans JP fonts
